# Versioned Chapter Results

These CSV files store the metric tables used to regenerate publication heatmaps:

```bash
python scripts/plot_result_heatmaps.py
```

The parameter counts were aligned with the packaged YOLOv11s definitions and verified saved checkpoints. For the DySample rows, the table reflects the stated two-stage neck design. GFLOPs are retained from the recorded chapter experiment logs.

Images, trained checkpoints, raw validation outputs, and generated plots are excluded from the repository.
